# Random_Align_Items 脚本 - Rank排列功能说明

## 新增功能：Rank排列

在原有的"齐头排列"和"随机排列"基础上，新增了"Rank排列"功能，专门用于处理带有rank标记的音效item的智能分布。

## Rank系统概述

Rank是你的自定义标记系统，用于区分不同类型的音效层级。例如：
- 科幻音效：glitch, click sci, sci imp, sci tonal
- 枪声音效：mech层, snap层, body层
- 情绪音效：low-frequency, high-frequency

## 使用流程

### 1. 定义Rank标记
首先使用 `Define_Item_Rank.lua` 脚本定义你的rank标记符号：
- 可以定义5个等级：笑脸1-3级、哭脸1级、空级
- 标记符号可以是任意字符，如 "asd", "glitch", "click" 等

### 2. 标记Item
在REAPER中，为需要分类的item添加notes（备注），在其中写入你定义的rank标记符号。

### 3. 使用Rank排列
1. 选中带有rank标记的items
2. 运行 `Random_Align_Items.lua`
3. 选择"Rank排列"模式
4. 选择分布方式：
   - **随机分布**：同rank的item可以随机混合，但尽可能均匀分布
   - **离散分布**：同rank的item尽量分散，避免同一时间轴上有两个相同rank
5. 调整集中度控制（与随机排列共享）
6. 点击"执行排列"

## 功能特点

### 智能识别
- 自动识别item的notes中的rank标记
- 支持自定义rank标记（任意文本）
- 未标记的item会单独处理

### 分布算法
1. **随机分布算法**：
   - 将所有item按rank分组
   - 随机打乱所有item顺序
   - 在保持rank多样性的同时随机分布
   - 避免item重叠

2. **离散分布算法**：
   - 为每个rank创建独立的时间槽
   - 同rank的item分配到不同的时间槽
   - 确保同一时间轴不会出现两个相同rank
   - 特别适合低频音效等需要离散分布的场景

### 输出信息
排列完成后会显示：
- Rank统计信息（每种rank的item数量）
- 分布方式
- 生成的组合数量

## 使用建议

### 科幻音效工作流
1. 定义rank：glitch, click, tonal, imp
2. 为每个科幻音效item添加对应的rank标记
3. 使用"离散分布"确保不同类型的科幻音效均匀分布
4. 调整集中度控制整体分布范围

### 枪声音效工作流
1. 定义rank：mech, snap, body
2. 标记枪声的各个组成部分
3. 使用"随机分布"创建自然的枪声组合
4. 通过多个组合生成丰富的枪声变化

## 注意事项

1. **标记一致性**：确保同一项目的rank标记符号一致
2. **未标记item**：没有rank标记的item会被放在最后随机排列
3. **集中度控制**：与随机排列共享，影响整体分布范围
4. **轨道分配**：每个item会分配到独立的轨道，便于后续处理

## 故障排除

### 问题：没有找到带有rank标记的Item
**解决**：
1. 确认已运行 `Define_Item_Rank.lua` 定义了rank标记
2. 检查item的notes中是否包含了定义的rank标记符号
3. 标记符号区分大小写，建议使用小写

### 问题：分布效果不理想
**调整**：
1. 尝试切换"随机分布"和"离散分布"
2. 调整集中度控制参数
3. 检查rank标记是否正确

### 问题：item重叠
**解决**：
1. 提高集中度控制的数值
2. 使用"离散分布"模式
3. 确保有足够的轨道空间
