# Random_Align_Items 脚本新功能说明

## 新增功能

### 1. Rank排列详细设置界面
在Rank排列界面中，新增了每种rank的排列细则控件，包括：

#### 数量限制功能
- 每种rank都可以设置最小和最大数量范围
- 默认范围：1~5个item
- 在排列时会随机选择该范围内的item数量
- 可以设置为0来完全禁用某种rank

#### 集中度控制按钮
- 每种rank都有独立的集中度控制滑块
- 控制范围：0.0（完全分散）到1.0（高度集中）
- 集中度影响：
  - 随机分布：高集中度使item更靠近中心位置
  - 离散分布：高集中度减少时间槽内的随机变化

### 2. Rank名称显示
- 自动从Define_Item_Rank脚本读取rank种类名称
- 显示格式：rank类型 + 用户定义的具体名称
- 例如："笑脸1级: good"、"哭脸1级: bad"

### 3. 界面布局改进
- 窗口大小增加到650x600以适应新控件
- 采用表格布局，每种rank占一行
- 每行包含：
  - 启用/禁用复选框
  - Rank名称显示
  - 数量限制输入框（最小~最大）
  - 集中度控制滑块
  - 集中度百分比显示

## 使用步骤

### 第一步：定义Rank
1. 运行Define_Item_Rank.lua脚本
2. 为5种rank类型定义具体名称
3. 配置会保存在当前工程中

### 第二步：标记Item
1. 在item的notes中添加相应的rank标记
2. 例如：在notes中输入"good"（对应笑脸1级）

### 第三步：使用Rank排列
1. 运行Random_Align_Items.lua脚本
2. 选择"Rank排列"模式
3. 在详细设置区域配置每种rank的参数：
   - 启用/禁用该rank
   - 设置数量限制范围
   - 调整集中度控制
4. 点击"执行排列"

## 技术实现

### 代码修改
1. **数据结构**：添加rank_settings表存储每种rank的配置
2. **GUI扩展**：在loop()函数中添加rank详细设置界面
3. **过滤逻辑**：在Rank排列执行前根据设置过滤和限制item数量
4. **分布算法**：修改两个分布函数以使用rank独立的集中度设置

### 关键变量
- `rank_settings`: 存储每种rank的详细配置
- `config.rank1_smile`等: 从Define_Item_Rank读取的rank名称
- `rank_display_names`: 用于界面显示的格式化rank名称

## 注意事项

1. 如果某种rank未在Define_Item_Rank中定义，界面会显示"未定义"
2. 数量限制不会超出实际可用的item数量
3. 集中度控制是独立的，不会影响其他rank
4. 禁用某种rank后，该rank的item将不会被排列

## 示例配置

假设Define_Item_Rank配置为：
- 笑脸1级: excellent
- 笑脸2级: good  
- 笑脸3级: ok
- 哭脸1级: bad
- 空级: none

在Rank排列界面中，可以这样配置：
- 笑脸1级: 启用，数量2~4，集中度0.8（高度集中）
- 笑脸2级: 启用，数量1~3，集中度0.5（中等集中）
- 笑脸3级: 启用，数量1~2，集中度0.3（较分散）
- 哭脸1级: 禁用
- 空级: 启用，数量1~5，集中度0.6（较集中）
