# Arrange_track_sample_GUI 新功能使用说明

## 新增功能：根据选中视频建立轨道

### 功能描述
当您在REAPER中选中一个或多个item（通常是视频文件）后，点击此功能按钮，脚本会自动：
1. 根据选中的item数量创建对应数量的一级轨道
2. 每个一级轨道下创建5个次级轨道
3. 一级轨道的名称与各个item的文件名一致（不含路径和扩展名）

### 使用步骤
1. 在REAPER中导入或创建一些视频item
2. 选中您想要创建轨道的视频item
3. 运行 `Arrange_track_sample_GUI.lua` 脚本
4. 在打开的GUI窗口中，找到"根据选中视频建立轨道"部分
5. 点击"根据选中视频建立轨道"按钮
6. 系统会自动创建轨道结构，并显示成功创建的消息

### 轨道结构示例
假设您选中了3个视频item，文件名分别为：
- `video1.mp4`
- `video2.mov`
- `video3.avi`

脚本将创建以下轨道结构：
```
Track 1: video1 (一级轨道)
  ├── video1_子轨道01 (次级轨道)
  ├── video1_子轨道02 (次级轨道)
  ├── video1_子轨道03 (次级轨道)
  ├── video1_子轨道04 (次级轨道)
  └── video1_子轨道05 (次级轨道)

Track 2: video2 (一级轨道)
  ├── video2_子轨道01 (次级轨道)
  ├── video2_子轨道02 (次级轨道)
  ├── video2_子轨道03 (次级轨道)
  ├── video2_子轨道04 (次级轨道)
  └── video2_子轨道05 (次级轨道)

Track 3: video3 (一级轨道)
  ├── video3_子轨道01 (次级轨道)
  ├── video3_子轨道02 (次级轨道)
  ├── video3_子轨道03 (次级轨道)
  ├── video3_子轨道04 (次级轨道)
  └── video3_子轨道05 (次级轨道)
```

### 注意事项
1. 确保您已经安装了ReaImGui扩展（脚本会自动检查）
2. 如果没有选中任何item，脚本会显示提示信息
3. 轨道名称会自动从文件名中提取，如果无法获取文件名，会使用"未命名"作为轨道名称
4. 每个轨道会自动分配不同的颜色以便区分
5. 创建轨道后，您可以使用REAPER的轨道管理功能进一步调整

### 与原功能的兼容性
新功能与原有功能完全兼容，您仍然可以使用：
- 手动添加和编辑轨道结构
- 创建自定义层级的轨道
- 调整轨道颜色和名称
- 使用原有的所有GUI功能

### 故障排除
如果新功能无法正常工作：
1. 检查是否选中了item
2. 确认item有有效的媒体源
3. 确保ReaImGui扩展已正确安装
4. 查看REAPER的控制台是否有错误信息
