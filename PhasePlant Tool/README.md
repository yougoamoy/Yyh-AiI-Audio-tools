# PhasePlant Tool

Kilohearts PhasePlant 合成器相关工具集。

## PhasePlant_Curve_Generator_GUI.py

### 功能描述
图形化界面工具,用于快速生成各种类型的PhasePlant包络曲线(.curve文件)。

### 主要特性
- 🎨 直观的图形界面
- 📊 实时曲线预览
- ⚡ 性能优化(减少75-90%的控制点)
- 🎲 支持随机变化(0-30%可调)
- 📦 批量生成(一次生成1-20个变体)
- 🎛️ 多种参数组合

### 参数说明

#### 1. 包络类型
- **一次性**: 适用于ADSR类包络
- **循环**: 适用于LFO类调制

#### 2. 曲线形状
- **线性**: 直线变化
- **指数**: 指数曲线(快速变化)
- **S曲线**: 平滑过渡

#### 3. 变化方向
- **上升**: 从低到高
- **下降**: 从高到低
- **起伏**: 波浪形

#### 4. 变化速度
- **快**: 8个控制点
- **中**: 12个控制点
- **慢**: 16个控制点

#### 5. 循环次数
- 范围: 1-16次
- 仅循环模式有效

#### 6. 随机变化
- 范围: 0-30%
- 在保持整体形状的同时添加细微差异

#### 7. 生成数量
- 范围: 1-20个
- 批量生成多个变体供选择

### 使用方法
```bash
python PhasePlant_Curve_Generator_GUI.py
```

### 输出位置
默认: `D:/Pan&Audio/预设/phaseplant/Curve/Generated/`
(可在界面中修改)

### 环境要求
- Python 3.6+
- tkinter (Python标准库,通常已包含)
