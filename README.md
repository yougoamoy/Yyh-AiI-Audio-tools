# 视频理解工具 - 音效制作助手

## 项目简介
这是一款专为音效工作者设计的视频理解工具，通过AI技术自动分析视频内容，在关键帧标注音效类型标签，辅助音效制作工作。

## 核心功能
- 🎬 视频关键帧自动提取
- 🤖 AI智能视频内容分析  
- 🔊 音效类型自动标记
- 💰 API费用实时监控
- 📊 处理日志详细记录

## 技术架构
- **AI模型**: 阿里云百炼 Qwen-VL 系列
- **视频处理**: OpenCV
- **用户界面**: Tkinter
- **费用管理**: 实时token消耗统计

## 使用场景
- 影视后期音效制作
- 游戏音频设计
- 广告配音配乐
- 教学视频音效添加

## 安装依赖
```bash
pip install opencv-python dashscope
```

## 使用方法
1. 运行 `python video_describer.py`
2. 选择要分析的视频文件
3. 等待AI分析完成
4. 查看生成的音效标签和描述

## 注意事项
- 需要阿里云百炼API密钥
- 建议使用qwen-vl-max模型获得最佳效果
- 大视频文件处理时间较长，请耐心等待
